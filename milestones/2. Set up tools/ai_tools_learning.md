## Which tool you used

I ended up installing Cursor. 

## What you used it for

I used it to help me debug my existing code (as shown above). As well as help explain what my code is doing so I have a better grasp on my projects. 

## What it helped with

It was really good at giving detailed enough answers and solutions for debugging. It's quite convenient that you can intergrate it with your terminal (terminal-friendly), which makes context switching seamless. 

## What it struggled with

The interface was very monotone, which makes it hard for me to concentrate sometimes. It also seems to struggle with large amounts of code. Even with the project I gave it (it had 170 lines of code), it took a minute to fully form an answer. So in foresight, I think this may struggle with even bigger and more complicated tasks.
