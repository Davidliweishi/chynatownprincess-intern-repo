1) Open Command Palette: ⌘ + ⇧ + P (Command + Shift + P)  - I'll use this alot to access the command palette easily without manually accessing it.

2) Quick File Open: ⌘ + P (Command + P)  - I can search up a file name quiclly in my repo.

3) Toggle Line Comment: ⌘ + / (Command + /) - Comment out a line.

4) F12 or ⌘ + Click (Command + Click) - jumps directly to the definition of a code (really handy for beginner developers) 

5) Format Document: ⇧ + ⌥ + F (Shift + Option + F) - helps format your code depending on the language you are using (like Prettier)

6) Deleting line (Command + shift + k ) - deletes line

7) Open Debug Panel: (Command + Shift + D) - open the Debug terminal 

8) Quick Fix: (Command + .): Show suggested actions for errors

9) F9: Toggle breakpoint - This helps check whether this line of code is executing properly or has any errors.

10) F5: Start/Continue Debug session
